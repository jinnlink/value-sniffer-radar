# Review Checklist

## Contract
- [ ] Scope matches the current ticket only
- [ ] Acceptance steps exist and are runnable
- [ ] Rollback plan exists

## Safety
- [ ] No secrets added (tokens/keys)
- [ ] No auto-trading added unless explicitly ticketed

## Quality
- [ ] Config changes documented
- [ ] New behavior has logs/observability
- [ ] Edge cases considered (rate limits, empty data, bad config)
